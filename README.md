#  AI FAQ Chatbot

An AI-powered FAQ chatbot developed using **Artificial Intelligence, Natural Language Processing (NLP), and Machine Learning** techniques.

This project uses **TF-IDF Vectorization and Cosine Similarity** to understand user questions and provide the most relevant answers from a predefined FAQ dataset.

Developed as part of the **CodeAlpha Artificial Intelligence Internship**.

---

#  Features

✅ Interactive chatbot user interface  
✅ Natural Language Processing (NLP)  
✅ Text preprocessing and cleaning  
✅ FAQ-based question answering  
✅ TF-IDF feature extraction  
✅ Cosine similarity matching  
✅ Intelligent answer prediction  
✅ Chat history support  
✅ Professional dark AI-themed interface  

---

#  Working Process

```
User Question
      |
      ↓
Text Preprocessing
      |
      ↓
Tokenization & Cleaning
      |
      ↓
TF-IDF Vectorization
      |
      ↓
Cosine Similarity Matching
      |
      ↓
Best FAQ Answer
      |
      ↓
Chatbot Response
```

#  Installation and Setup

## Clone Repository

```bash
git clone https://github.com/kaviya050102/CodeAlpha_FAQ_Chatbot.git
```

## Navigate to Project Folder

```bash
cd CodeAlpha_FAQ_Chatbot
```

## Install Required Libraries

```bash
pip install -r requirements.txt
```

## Run Application

```bash
streamlit run app.py
```
